# Slack Themes (WIP)

> Minimal & Modern React app documenting a collection of Slack Themes

---

## Usage
```bash
# Install dependencies

 $ npm install
 
# Start webpack-dev-server at port 8080

 $ npm start
 
# Run linters

 $ npm test
 
# Build app for production (gets output in the 'dist' directory)

 $ npm run build
```
